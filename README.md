# R-ATC
R-ATC(ATACS) Plug-in for BVE5 by Stop_Pattern  


R-ATCを疑似再現するプラグインです  

TetsuOtter氏の[BVE_ATSPITemp](https://github.com/TetsuOtter/BVE_ATSPITemp)をもとに作られています
ご使用の際はソースを読んで地上子を置く鬼畜仕様ですので分かる方だけどうぞ  
もちろんサポートなんてありません
