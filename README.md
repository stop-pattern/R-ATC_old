# R-ATC
R-ATC(ATACS) Plug-in for BVE5 by Stop_Pattern  

## feathers
R-ATCを疑似再現するプラグインです  
ATC-6互換機能を搭載しているため、ATC-6路線も走行可能です。  
ご使用の際はソースを読んで地上子を置く鬼畜仕様ですので分かる方だけどうぞ  

## environment
- OS：Windows10  
- Platform：BVETrainSim v5.7
